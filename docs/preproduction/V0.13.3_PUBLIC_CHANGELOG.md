# Smash & Lob v0.13.3 · Registro público de cambios

## Alcance

- Nueva pantalla `/changelog` con el historial público de versiones documentadas.
- Acceso desde Ajustes para jugadores y espectadores.
- El número de versión visible en Ajustes enlaza directamente al historial.
- El buscador de Ajustes incluye el nuevo acceso en castellano, inglés y euskera.
- La versión actual queda destacada y las revisiones menores pueden aparecer agrupadas cuando no existe un detalle público fiable de cada parche.

## Privacidad

El contenido público excluye credenciales, datos personales, identificadores internos, configuración de infraestructura, detalles de despliegue y procedimientos de seguridad.

## Base de datos

No requiere migraciones ni cambios de datos.

## Validación recomendada

1. Abrir Ajustes como jugador y acceder a Registro de cambios.
2. Pulsar el número de versión del pie de Ajustes y comprobar que abre la misma pantalla.
3. Buscar `cambios`, `versiones` o `changelog` en el buscador de Ajustes.
4. Abrir la pantalla como espectador y comprobar que no redirige a HOME.
5. Desplegar y cerrar varias versiones.
6. Revisar la visualización en móvil, modo oscuro y PWA.
7. Ejecutar `npm run lint`, `npx tsc --noEmit` y `npm run build`.

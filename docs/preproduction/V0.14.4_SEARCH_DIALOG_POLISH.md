# v0.14.4 - Pulido del buscador flotante

Fecha: 24 de julio de 2026

## Alcance

Esta revisión modifica únicamente la presentación del buscador global de Ajustes. No cambia el índice de resultados, las rutas, los permisos, las APIs ni la base de datos.

## Cambios

- El panel se coloca con el borde superior fijo, justo debajo de los controles flotantes superiores de la aplicación.
- La altura se adapta al número de accesos o resultados mostrados.
- La posición vertical no cambia al escribir ni al variar la cantidad de resultados.
- El contenido obtiene desplazamiento interno únicamente cuando supera la altura máxima disponible.
- La altura máxima respeta el viewport estable, las áreas seguras y la navegación inferior.
- El fondo con blur se extiende por encima del viewport visible para evitar una franja superior sin desenfoque en dispositivos móviles.

## Verificación manual

1. Abrir el buscador sin texto y comprobar su posición bajo los controles superiores.
2. Buscar una opción con un único resultado y verificar que el panel se compacta hacia abajo.
3. Buscar una consulta con muchos resultados y verificar que el borde superior no se mueve.
4. Confirmar que solo la lista de resultados obtiene scroll al superar la altura máxima.
5. Abrir y cerrar el teclado móvil y comprobar que el panel no salta de posición.
6. Verificar que el blur cubre completamente el borde superior del dispositivo.
7. Revisar modo claro y oscuro.

## Base de datos

No requiere migraciones de Supabase.
